# Concert Nature

Site web moderne pour des événements musicaux en plein air, construit avec HTML, CSS et JavaScript, Hugo et déployé sur Netlify.

git submodule add zhaohuabing/hugo-theme-cleanwhite  themes/hugo-theme-cleanwhite

## Configuration technique

- **CMS** : Netlify CMS (Headless)
- **Hébergement Git** : GitHub
- **Déploiement** : Automatique avec Netlify
- **Domaine** : www.concertnature.com
- **Générateur de site** : Site statique HTML/CSS/JS

## Fonctionnalités

- Design responsive qui s'adapte à tous les appareils
- Animations fluides et interactives
- Navigation fluide avec défilement
- Optimisé pour les performances
- Déploiement continu avec Netlify

## Structure du projet

```
website-concertnature/
├── index.html          # Page d'accueil
├── styles.css          # Feuilles de style
├── script.js           # JavaScript pour les interactions
├── netlify.toml        # Configuration Netlify
└── readMe.md           # Ce fichier
```

## Développement local

1. Clonez ce dépôt
2. Ouvrez le fichier `index.html` dans votre navigateur

## Déploiement

Ce site est configuré pour un déploiement automatique sur Netlify. Pour déployer :

1. Poussez votre code sur GitHub
2. Connectez votre dépôt à Netlify
3. Le déploiement se fera automatiquement à chaque push

## Personnalisation

- Modifiez les couleurs dans `styles.css` (variables CSS en haut du fichier)
- Mettez à jour le contenu dans `index.html`
- Ajoutez vos propres images dans un dossier `images/`
